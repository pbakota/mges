# MGES with Python3

## Installing dependencies

```
python -m pip install pysdl2
```

or

```
sudo apt install python3-sdl2
```

## Run the game
```
python3 main.py
```


## Tools used

### For developing

```
VSCode
```

### For replace import * with names
```
pip install dewildcard
```

### For formating
```
python3 -m pip install git+https://github.com/psf/black
```
